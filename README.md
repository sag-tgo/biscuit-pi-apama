# biscuit-pi-apama
A biscuit sorting robot - Lego Mindstorms EV3, BrickPi3, Python, Apama EPL

# Hardware requirements
- LEGO® MINDSTORMS® Education EV3 Core Set (45544)
- Raspberry Pi 3 Model B
- Dexter BrickPi3 Base Kit
